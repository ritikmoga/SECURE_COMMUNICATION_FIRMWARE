# Project Screenshots

These images are presentation-ready concept mockups for the planned Secure Communication Firmware dashboard. They illustrate the intended dashboard, encrypted chat, connected-device view, certificate management, web logs, relay-server logs, OTA-server logs, and device CLI workflow.

They are not captures of a currently deployed browser dashboard. The reference implementation currently provides ESP32-S3 firmware, a mutual-TLS relay, command-line interaction, and an HTTPS OTA file server.
